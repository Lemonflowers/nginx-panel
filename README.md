基于 yum apt dnf 安装的nginx开源 反向面板<br>
反向代理 集成 HTTP2 WS GZIP 自定义缓存<br>
四层转发 支持TCP/UDP<br>
系统监控 实时宽带 累计流量 实时CPU 实时内存 实时磁盘<br>
ACME集成 支持域名签证 域名自动续签<br>
cloudfare 集成 支持DNS 全局API管理<br>
图床模块 内置图床应用 开箱即用<br>
日志管理<br>
日志分析（基于PHP）<br>
缓存管理 （文件）<br>
5秒盾 （请求频率）<br>
IP拉黑 全局模式<br>
防跨站 （SSL）<br>

centos7.6-7.9<br>
